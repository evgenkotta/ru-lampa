# 🇷🇺 RU Lampa

Русификатор интерфейса для приложения Lampa.

## Возможности

- 🇷🇺 Перевод интерфейса на русский язык
- 🔄 Простая установка через ссылку на плагин
- ⚡ Автоматическая загрузка обновлений через GitHub Pages

## Установка

1. Откройте Lampa.
2. Перейдите в:
   Настройки → Плагины → Добавить плагин
3. Вставьте ссылку:

https://evgenkotta.github.io/ru-lampa/rus-actual.js

4. Перезапустите Lampa.

## Разработка

Репозиторий использует GitHub Pages для публикации актуальной версии плагина.
