# RU Lampa

Плагин для Lampa, который добавляет на главную страницу отдельные ряды русских фильмов и сериалов, доступных для онлайн-просмотра.

## Возможности

* Отдельный ряд «Русские фильмы»
* Отдельный ряд «Русские сериалы»
* Автоматическое размещение рядов первыми на главной странице
* Подбор только проектов, доступных для онлайн-просмотра
* Исключение мультфильмов (по желанию)
* Настройки прямо в меню Lampa
* Поддержка Web, Android TV и webOS
* Использование базы TMDB

## Установка

Добавьте в Lampa следующий адрес:

```text
https://evgenkotta.github.io/ru-lampa/rus_actual.js
```

## Как работает

Плагин использует TMDB Discover API и автоматически формирует подборки:

* только российских проектов (`with_origin_country=RU`);
* только русскоязычных проектов (`with_original_language=ru`);
* только проектов, доступных для онлайн-просмотра в регионе RU (`watch_region=RU` и `with_watch_monetization_types=flatrate|free`);
* без мультфильмов по умолчанию (`without_genres=16`);
* отсортированных по дате выхода — сначала самые новые релизы.

## Настройки

Плагин позволяет:

* включать и отключать ряд «Русские фильмы»;
* включать и отключать ряд «Русские сериалы»;
* скрывать или показывать мультфильмы;
* ограничивать количество отображаемых карточек.
