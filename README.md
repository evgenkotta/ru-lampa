# RU Lampa

Плагин для Lampa, который добавляет на главную страницу отдельные подборки русских фильмов и сериалов.

## Возможности

* Отдельный ряд «Русские фильмы»
* Отдельный ряд «Русские сериалы»
* Размещение рядов в начале главной страницы
* Исключение мультфильмов
* Настройки прямо в меню Lampa
* Поддержка Web, Android TV и webOS
* Использование базы TMDB

## Установка

Добавьте в Lampa следующий адрес:

```text
https://evgenkotta.github.io/ru-lampa/rus_actual.js
```

При обновлении плагина рекомендуется использовать параметр версии:

```text
https://evgenkotta.github.io/ru-lampa/rus_actual.js?v=2
```

## Настройки

Плагин позволяет:

* включать и отключать фильмы;
* включать и отключать сериалы;
* скрывать или показывать мультфильмы;
* отображать только русскоязычные проекты;
* выбирать сортировку;
* задавать минимальное количество голосов;
* ограничивать количество отображаемых карточек.
